Issue Tracking

The Issue Tracking system for the Coherence Hibernate Integration project is hosted by GitHub.

It can be found here: https://github.com/coherence-community/coherence-hibernate/issues

Raising Issues:

To raise an issue representing a defect or feature request in the issue tracking system you need to have a GitHub account.

Modifying and Closing Issues:

To resolve and close issues with the issue tracking system you will need to be a contributor.
